Stream Gage Data Providers

This is a list of the data providers used by HydroCloud. Click on the links below to query each provider for a downloadable list of available stream gage sites.

We are always looking for more data providers! If you know of a data service that provides access to stream discharge measurements, let us know by filing an issue on GitHub.

Data Providers:

  • USGS Daily (USA): Download CSV of stations
  • USGS Real-time (USA): Download CSV of stations
  • PEGEL-ONLINE (Germany): Download CSV of stations
